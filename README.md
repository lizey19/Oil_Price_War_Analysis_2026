# 🛢️ 2026 Iran-War Oil Shock Analysis

## 📖 Overview
An advanced data science study on the global energy crisis of 2026. This project quantifies the "War Premium" on Brent Crude and simulates future price risks using Monte Carlo methods.

## 🛠️ Tech Stack
- **Python** (Pandas, NumPy, Scikit-Learn)
- **Statistical Modeling** (ARIMA, GARCH, Causal Inference)
- **Machine Learning** (K-Means Clustering)

## 🚀 Key Findings
- Isolated a **$33.15 War Premium** per barrel.
- Identified **95% Value at Risk (VaR)** at $138.40.
- Clustered US states by economic vulnerability to gas price shocks.
